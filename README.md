# Hamza Ali Developers - Company Website

A professional single-page website for Hamza Ali Developers, an IT services company specializing in mobile application development.

## Features

- **Modern Design**: Clean, professional, and responsive design
- **Mobile Responsive**: Works perfectly on all devices (desktop, tablet, mobile)
- **Smooth Scrolling**: Smooth navigation between sections
- **Contact Form**: Interactive contact form for inquiries
- **Play Store Ready**: Suitable for Play Store and App Store verification

## Files Structure

```
hamza-ali-developers/
├── index.html      # Main HTML file
├── styles.css      # Styling and design
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## How to Use

1. **Local Development**: Simply open `index.html` in a web browser
2. **Web Hosting**: Upload all files to your web hosting server
3. **Customization**: 
   - Update your email + WhatsApp number in `index.html`
   - Modify colors and spacing in `styles.css`
   - Update services/content as needed

## Customization

### Change Colors
Edit the theme values in `styles.css` (search for the gradient/background colors).

### Update Contact Information
Edit the contact section in `index.html` to update email and business hours.

### Add Your Logo
Replace the text logo in the navbar with an image if you have one.
